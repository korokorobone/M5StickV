# M5StickV

※トライスタ技術　芸カ20号のソフトウェア入れています。  
「boot1」のフォルダ内のデータをmicroSDカードに入れてください。  
なお、ソースコードは予告なく変更することがあります。  
 「ポン」(物音11)と「キラリン」(マジカル25)という効果音はフリー音源です。  
 ( フリー音源元：魔王魂　https://maoudamashii.jokersounds.com/ )
